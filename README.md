      二进制数又分为正数和负数 ，区分的方式就是符号位是0还是1  0就表示正数 1就表述负数  （这里的符号位是指从右往左数的第8位）
      二进制正数如何转换为负数：
               例子:                    0 0 0 0 1 1 1 0        （14）
               第一步: 符号位变为1       1 0 0 0 1 1 1 0        (称为 -14的原码）
               第二步：除符号位外取反    1 1 1 1 0 0 0 1        （称为-14的反码）
               第三步：反码加 1          1 1 1 1 0 0 1 0        （称为-14的补码）
                得出的补码即为某整二进制数的对应负数
